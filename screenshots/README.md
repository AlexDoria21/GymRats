# Screenshots

Coloca aquí las capturas de pantalla de la app. El `README.md` principal las referencia
con estos nombres (renombra tus archivos para que coincidan, o ajusta el README):

| Archivo               | Pantalla                                  |
| --------------------- | ----------------------------------------- |
| `home.png`            | Inicio: lista de rutinas + calendario     |
| `routine.png`         | Rutina: días con drag & drop              |
| `day.png`             | Día: ejercicios, series, biseries         |
| `rest-timer.png`      | Temporizador de descanso                  |
| `progress.png`        | Gráfica de progreso + 1RM                 |
| `settings.png`        | Ajustes (unidad, notificaciones, datos)   |

**Recomendado:** capturas en formato vertical (móvil), PNG, ~1080px de ancho.
